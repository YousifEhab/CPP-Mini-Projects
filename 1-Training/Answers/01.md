## 1- Return Negative.

### Ask the user to enter a positive integer and the system prints the negative value of the same integer.

```cpp
#include <iostream>
using namespace std;
int main()
{
    cout << "enter Integer number ";
    int Number;
    cin >> Number;
    cout << "The negative value of " << Number << " is " << Number * -1;
    // system("pause");
    return 0;
}
```
