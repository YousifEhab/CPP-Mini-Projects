## 10-
### Write a program to read any day number in integer and display the day name in word format.
```cpp
#include <iostream>
using namespace std;
int main()
{
    int x;
    cout << "Enter The Number Between 1 and 7" << endl;
    cin >> x;
    switch (x)
    {
    case 1:
        cout << "Sunday" << endl;
        break;
    case 2:
        cout << "Monday" << endl;
        break;
    case 3:
        cout << "Tuesday" << endl;
        break;
    case 4:
        cout << "Wednesday" << endl;
        break;
    case 5:
        cout << "Thursday" << endl;
        break;
    case 6:
        cout << "Friday" << endl;
        break;
    case 7:
        cout << "Saturay" << endl;
        break;
    default:
        cout << "error message." << endl;
    }
    // system("pause");
    return 0;
}
```
