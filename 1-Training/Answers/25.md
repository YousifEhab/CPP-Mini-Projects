## 25 - Get Nth Even Number
### Return the Nth Even Number
```
1 --> 0 (the first even number is 0)
3 --> 4 (the 3rd even number is 4 (0, 2, 4))
100 --> 198
1298734 --> 2597466
```
```cpp

#include <iostream>
using namespace std;
int main()
{
    int n, x;
    cout << "Enter num" << endl;
    cin >> n;
    if (n < 1)
    {
        cout << "Invalid Input\n";
        return 0;
    }
    x = (n-1)*2; 
    cout << "Nth Even Number is = " << x << endl;
    // system("pause");
    return 0;
}
```
